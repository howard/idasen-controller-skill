Hack the desk

  1. Install client: 
    pip install idasen-controller
  2. Pair device with desk
  3. Configure the client: 
    # ~/.config/idasen-controller/config.yaml
mac_address: XX:XX:XX:XX:XX:XX
favourites:
  stand: 1105
  sit: 737
  4. Make desk go bzzzzt: 
    idasen-controller --move-to stand
